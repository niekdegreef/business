Process

  1. Design & set up styles in AI
  2. Configure servers: MAMP / remote dev server [check php versions are same]
  3. Set up folder structure, drop into codekit [check libsass comiler & autoprefixr checked]
  4. Download latest underscores starter theme: http://underscores.me/
  5. Get latest version of susy https://github.com/ericam/susy (or use via Codekit)
  6. Include “ @import "susy”; “ in the main.scss to use Susy via Codekit
  7. Set basic variables, colours, padding, margin (with susy so these are relative to the grid)
  8. Edit the styles document, change colour names & classes where needed
  9. Define typography (http://www.gridlover.net/app/ & http://typecast.com/)
  10. Take parts from the underscores.css and put them in correct files
